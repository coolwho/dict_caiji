搜狗百科：

install

只要计算机开着，内存驻留程序就一直在内存中，甚至当其它程序活动时，还能使用内存驻留程序，用INSTALL命令可以装入MS-DOS内存驻留程序，例如Fastopen，keyb、Nlsfunc和Share。此命令不为装入的程序生成环境，只能在CONFIG.SYS文件中使用。
