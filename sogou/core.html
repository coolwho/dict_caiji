单词辨析

  • 中心

    • midst 书面语用词，指在一个群体的深处或在某活动的进程中。
    • center 指三维空间的中心点，也可比喻抽象事物的中心。
    • heart 指事物最内部或最重要的部分，表地理位置时可与center换用。
    • core 指某事物固定的中心部分或最重要的核心部分。比喻意义指某物的精华。
    • middle 一般指时间、空间或过程两端间等距的部分。

core

core，英文单词，主要用作名词、及物动词，作名词时译为核心；要点；果心；[计] 磁心；(Core)人名；(英)科尔；(西、意)科雷。作及物动词时译为挖...的核。

core

Core：特尔处理器的名称。服务器版的开发代号为Woodcrest，桌面版的开发代号为Conroe，移动版的开发代号为Merom。分双核、四核、八核三种。酷睿处理器采用800MHz-1333Mhz的前端总线速率，45nm/65nm制程工艺，2M/4M/8M/12M/16M L2缓存，双核酷睿处理器通过SmartCache技术两个核心共享12M L2资源。
