短语搭配

  • by default

    (比赛)因其他参赛者不到场, 由于对手缺席(而胜出)

  • in default

    拖欠违约；不出庭罪

  • in default of

    因缺少；在缺乏…时；由于没有协议；租金将由房产稽查员决定

  • go by default

    因一方未提出反对而判另一方获胜

  • judgement in default

    缺席审判

Default

default，英文单词，可作动词、名词，译为未履行任务或责任、受传唤时未出庭、由于不到庭而败诉、弃权、拖欠等。

Default
